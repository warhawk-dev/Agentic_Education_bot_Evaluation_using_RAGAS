Before running the project, make sure that you have to create 'uploads' and 'chroma_db' folder so as to store the uploaded PDFs
Also create .env file for storing APIs, it should contain:

GROQ_API_KEY    = 'Your API Key'
LANGCHAIN_ENDPOINT = 'https://api.smith.langchain.com'
LANGCHAIN_TRACING_V2=true
LANGCHAIN_API_KEY = 'Your API Key'
